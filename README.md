# Homebrew Tap for Codeep

Official Homebrew tap for [Codeep](https://github.com/VladoIvankovic/Codeep) - AI-powered coding assistant for the terminal.

## Installation

```bash
brew tap VladoIvankovic/codeep
brew install codeep
```

## Usage

After installation, run:

```bash
codeep
```

## Updating

```bash
brew update
brew upgrade codeep
```

## Uninstall

```bash
brew uninstall codeep
brew untap VladoIvankovic/codeep
```
